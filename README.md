# Data-Champion-Growth-Analyst-Assignment

**Overview**

This repository contains my submission for the **Growth Analyst Task (Deep Thought)**.
The objective of this assignment was to demonstrate how AI prompting, funnel diagnostics, and data-driven decision-making can be combined to improve B2B outbound performance.

The work focuses on using AI strategically — not just for content generation, but for **conversion optimization, personalization at scale, and funnel-level problem solving.**

**Part Breakdown**

**Part 1 — Prompt Engineering for Mass Personalization**

Designed AI prompt frameworks using:

AIDCA conversion model

Cialdini persuasion principles

Persona-based targeting

Goal: Enable scalable, high-quality outbound personalization.

**Part 2 — Funnel Debugging via Prompt-Based Diagnosis**

Simulated campaign funnel data and analyzed:

Message–Market Fit (MMF)

Funnel drop-off patterns

Root causes of conversion loss

**Part 3 — Dashboard Design with Boardroom Intent**

Designed a leadership-focused dashboard wireframe to track:

Funnel health

Persona performance

Campaign-level AIDCA breakdown

Strategic intervention signals

Goal: Convert metrics into actionable decisions.

**Part 4 — Strategic Summary**

Summarized growth mindset, AI usage philosophy, and iterative optimization approach in under 200 words.
Created targeted AI prompts to fix specific funnel-stage failures.
